## lua

先说lua程序的执行流程，超级简单

从第一行开始读，读到什么就做对应的事，按顺序一句一句执行过去，没了

所有句子结构：

| 结构          | 格式                                   | 效果                                 | 例 & 说明                                                                                                                     |
| ------------- | -------------------------------------- | ------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| 表达式        | 一堆数字（或变量）通过运算符连接在一起 | 计算得到新的值                       | 虽然单独的表达式不能作为独立的语句，但其他语句中都需要填表达式                                                                |
| 执行函数      | 函数(参数1, ...)                       | 触发一套别人打包好的一系列功能       | `print("Hello")`会调用print函数并传入一个字符串`Hello`作为参数，它的功能就是把输入的东西显示到控制台里，于是看到Hello五个字母 |
| 赋值语句      | 变量名 = 表达式                        | 给变量存入一个值（或是要计算的式子） | 执行`a=1`，此时变量a中就存储了数字1，然后执行`b=a+1`，等号右边的表达式计算结果是2，存入变量b                                  |
| if控制语句    | if 表达式 then 语句 else 语句 end      | 根据条件执行不同的语句               |                                                                                                                               |
| while控制语句 | while 表达式 do 语句 end               | 根据条件反复执行一段代码             |                                                                                                                               |
| for控制语句   | for 变量名 = 初值, 终值 do 语句 end    | 执行固定次数的语句                   |                                                                                                                               |
| 定义函数      | function 变量名(参数1, ...) 语句 end   | 打包一段代码方便之后多次使用         |                                                                                                                               |

所有运算符：

| --------- | ------------ | ---------------------------------------- |
| + - * /   | 加减乘除     | 略                                       |
| %         | 取余         | 略                                       |

刚刚发的学习包里有一个运行a.lua.bat，双击就会用lua.exe打开你写的a.lua并一行行执行

现在开始只要关心a.lua里写了什么就行，bat只作为一个启动器（不然你每次都要cmd里敲lua.exe a.lua来执行你写的东西，太累了）

变量只有这几种类型：数字 字符串 布尔值 表 函数

前三个简单，表是一个“装东西”的东西，基础用法跟序列（列表）一样

```lua
    a={}
    a[1]=20
    a[2]=26
```

意思就是a是一个表，然后“第一个”是数字20，“第二个”是数字26

强大的来了，谁说“第几个”的“几”一定要是数字？我们甚至可以：

```lua
    a={}
    a["b"]=200 --第"b"个是200
    a["你好"]="世界" --第"你好"个是"世界"
    a[true]=false --第true个是false
```

神奇吧！所以最后的效果就是你可以把 `a["你好"]` 这样格式东西视为一个变量操作

补充一下，因为 `a["b"]` 这种格式用的很多很多，所以lua提供了一个“缩写”，写成a.b就可以了

## 游戏程序的基本结构

```lua
    while true do
        if 按了鼠标 then 鼠标事件() end
        if 按了键盘 then 键盘事件() end
        （其他各种各样的操作，触摸屏什么的）

        固定做一些事情

        在窗口里画东西
        等1/60秒
    end
```
